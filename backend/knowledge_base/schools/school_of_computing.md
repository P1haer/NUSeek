# School of Computing, National University of Singapore

## 学院概况
- **成立年份**: 1981
- **世界排名**: QS 2024 前10
- **学生人数**: 约3,200人（本科）+ 2,000人（研究生）
- **教职员工**: 250+位教授和讲师
- **校区位置**: Kent Ridge Campus, Blk COM1

## 本科课程（Bachelor of Computing）

### 核心课程（第1-2年）
**第一学期:**
- CS1101S Programming Methodology (4 MCs)
- CS1231 Discrete Structures (4 MCs)
- MA1101R Linear Algebra I (4 MCs)
- MA1102R Calculus (4 MCs)

**第二学期:**
- CS1102S Data Structures and Algorithms (4 MCs)
- CS2100 Computer Organisation (4 MCs)
- MA1521 Calculus for Computing (4 MCs)
- CS2101 Effective Communication for Computing Professionals (4 MCs)

### 专业方向（3年级开始选择）
1. **Artificial Intelligence & Machine Learning**
   - CS3244 Machine Learning
   - CS3263 Foundations of Artificial Intelligence
   - CS4243 Computer Vision
   - CS4248 Natural Language Processing

2. **Computer Security**
   - CS3235 Computer Security
   - CS4236 Cryptography Theory and Practice
   - CS4261 Cyber-Physical System Security

3. **Database Systems & Data Analytics**
   - CS2102 Database Systems
   - CS3223 Database Systems Implementation
   - CS4221 Advanced Data Analytics

4. **Software Engineering**
   - CS2103T Software Engineering
   - CS3213 Software Engineering Project
   - CS4218 Software Testing

5. **Computer Networks**
   - CS2105 Introduction to Computer Networks
   - CS3103 Computer Networks Practice
   - CS4344 Networked and Distributed Systems

### 毕业要求
- **总学分**: 160 MCs (Modular Credits)
- **最低GPA**: 2.0/5.0
- **核心课程**: 80 MCs
- **专业课程**: 40 MCs
- **通识教育**: 20 MCs
- **毕业设计/论文**: 20 MCs（可选）

### 就业前景
- **毕业平均起薪**: SGD $4,500 - $6,500/月
- **主要雇主**: Google, Meta, Microsoft, Amazon, ByteDance, Grab, SEA
- **升学率**: 约40%（硕士或博士）
- **职业方向**: 
  - Software Engineer
  - Data Scientist
  - Product Manager
  - Solutions Architect
  - Security Analyst

## 研究生课程

### Master of Science in Computer Science (MSc CS)
**课程类型**: 
- Thesis Track (以研究为主)
- Coursework Track (以课程为主)

**申请要求**:
- 本科学位（任何专业）
- GPA: 3.0/4.0 或同等
- TOEFL: 92 (iBT) 或 IELTS: 6.5
- GRE 推荐但非必需

**学期长度**: 2-3年

**热门专业方向**:
- Artificial Intelligence
- Computer Networks
- Computer Security
- Database Systems
- Software Engineering
- Bioinformatics

### Ph.D. in Computer Science
**申请要求**:
- 相关领域硕士学位或本科一等荣誉
- 研究兴趣说明书
- 3封推荐信
- TOEFL/IELTS 成绩

**资助**:
- 全额奖学金（学费 + 生活费 SGD $2,000/月）
- 许多学生可获得 Research Scholarship

**完成时间**: 4-5年
